# v76-ux-readability-tuning-2026-07-18

## 版本重點

- 玩家桌面籌碼從 `70%` 高度顯示，遠離自己的手牌區，降低重疊感。
- 操作按鈕預設位置上移，並透過新版 layout key 重新套用安全座標。
- `跟注/過牌` 改成更亮的綠色，`加注` 改成金色，讓主要操作不像 disabled。
- 已棄牌玩家與蓋牌區淡化，桌面焦點更集中在仍參與牌局的玩家。
- 當前行動玩家加入呼吸 glow，辨識度更高。
- 牌局紀錄增加字級、行高與 padding，讀起來不那麼擠。
- 已同步到 `docs/`，給 GitHub Pages 使用的檔案內容一致。
