# v51-keyboard-layout-nudge-2026-07-12

## 版本註解

讓版面編輯器支援鍵盤方向鍵微調。

## 改動

- 編輯版面開啟時，可以用鍵盤方向鍵微調目前選取元素。
- 方向鍵每次移動 0.5%。
- `Shift + 方向鍵` 每次移動 2%，適合快速粗調。
- 鍵盤微調只在版面編輯模式生效。
- 鍵盤微調不會攔截輸入框、滑桿、下拉選單或可編輯文字區。
- 編輯面板新增提示文字：
  - 方向鍵微調
  - Shift + 方向鍵加速
- 面板狀態文字會顯示目前選取元素座標，例如 `(61.5%, 90.0%)`。
- 同步更新 `docs/` 部署用檔案。
- 保持三個主檔結構，沒有拆出新的 JS/CSS 子檔。

## 驗證

- `node --check app.js`
- `node --check docs/app.js`
- 根目錄與 `docs/` 的 `index.html`、`styles.css`、`app.js` 已同步。
- 本機瀏覽器預覽通過：
  - 編輯版面可開啟。
  - 選取玩家資訊欄後，右方向鍵從 `61%` 微調到 `61.5%`。
  - `Shift + 下方向鍵` 從 `88%` 快速微調到 `90%`。
  - 狀態文字會同步顯示新座標。
  - Console 無 error/warning。
