# v77-pixel-card-theme-2026-07-19

## 版本重點

- 牌面改成乾淨像素牌組風格，接近使用者提供的 light/dark/sepia card set 參考。
- 數字牌保留清楚角標與花色排列，降低過度裝飾造成的辨識成本。
- J/Q/K 改成簡化像素人物牌面，不再只是大花色。
- 牌背改成簡潔幾何紋樣，與參考圖的卡背方向一致。
- 明亮模式移除上一版咖啡色復古牌面覆寫，與暗色模式共用同一套牌組語言。
- 快取版本號更新為 `session-summary-v77`。
- 已同步到 `docs/`，給 GitHub Pages 使用的檔案內容一致。
